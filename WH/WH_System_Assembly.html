Dashboard Guide

1. Filter Heatmap by Build Date

Use "Ctrl + Click" (Windows) or "Cmd + Click" (Mac) to select multiple build dates. This updates the heatmap to show cumulative failures only for the selected builds.

2. Heatmap Types

  • UBB vs ASIC: Shows failures mapping specific UBB slots to ASIC locations.
  • UBB vs ETH: Shows network failures per UBB slot.
  • UBB vs Ctrl: Shows controller failures per UBB.
